# **App Name**: MICONDO APP

## Core Features:

- Dashboard: Display a list of condominiums with key information and a button to manage each one.
- CondoAdminPanel: Provide a collapsable sidebar navigation with tabs for Finances, Units, Accounts Receivable, Accounts Payable, Suppliers, Employees, Communications, Incidents, and Reports.
- Finances Tab: Show KPIs, a debt summary, a financial chart, and a transaction history table with balance calculation. Allows users to register new transactions via a modal.
- Units Tab: Display a table listing all units with options to add and edit units using a detailed modal form.
- Accounts Receivable Tab: Provide a table showing the current account balance for each unit, with a modal for detailed account history and management options.
- Accounts Payable Tab: List all invoices, highlighting overdue dates in red. Enable invoice PDF generation, payment processing, editing, and deletion.
- Suppliers and Employees Tabs: Implement a two-level view for suppliers and employees, allowing users to manage invoices and payroll information, respectively. Suppliers have invoice history, employees have detail and payroll settings views. Provide importing function to onboard suppliers from a CSV file.

## Style Guidelines:

- Primary color: A calm blue (#5072A7), evoking trust and stability.
- Background color: Light grey (#F0F2F5), providing a neutral backdrop that enhances content visibility.
- Accent color: Soft green (#8EC077) for CTAs and highlighting key interactions.
- Headline font: 'Poppins', sans-serif, for a precise, contemporary feel; body font: 'PT Sans', sans-serif, for comfortable reading.
- Lucide icons will be used throughout the application, maintaining consistency and visual appeal.
- Responsive design with a collapsable sidebar for optimal viewing on all devices. Use of Card components for clear information display.
- Subtle animations on modal transitions and data updates to improve user experience.
